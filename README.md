# Biblioteca datetime en Python

Trabajo académico sobre la biblioteca `datetime` de Python.

Incluye:

- Explicación de las clases principales (`date`, `time`, `datetime`, `timedelta`)
- Funciones importantes (`strftime()`, `strptime()`)
- Ejemplo práctico en Python
- Documento en LaTeX con desarrollo teórico

## Estructura

- `src/` → Código del ejemplo práctico  
- `docs/` → Documento en LaTeX
